Ubuntu系统安装教程

一. 前言

本篇文章详解介绍一下如何安装Ubuntu系统，笔者在安装的过程中踩过很多坑，重装了很多次，现在把安装过程中遇到的问题也列出来，供大家参考。

二. 准备工作

这个环节很重要，工欲善其事，必先利其器。

2.1 固态硬盘

如果是做系统源码开发的话，买一块内存大于 512G的SSD，如果经济条件好的话，容量越大越好。

2.2 UEFI + GPT 安装

安装之前，查看一下自己电脑的BIOS系统的类型，使用快捷键 win + R 输入 msinfo32。

如图：

如上图所示，在右边找到“BIOS模式”，看是否为“UEFI”。

2.3 制作启动盘

使用 rufus工具制作启动盘。

2.4 电脑设置

以笔者的电脑举个例子，型号:联想E430c。
设置 UEFI/Legacy Boot选项 为UEFI，如图：

关闭secure boot 设置成disable：

USB启动为enable 目的：可以通过USB盘启动系统，如下图：
进入StartUp -> Boot：

光标移动到USB选项上，然后按“+”号将USB移动到第一行，从USB启动系统。

三. 普通安装

3.1 解决卡在logo界面

通过USB系统盘启动后，进入到如下界面：

这里会有一个坑，Ubuntu启动时卡在logo页面。解决方法：移动到 Install Ubuntu 此项上，然后输入 e （edit模式）。
修改 quiet splash --- 为 quiet splash nouveau.modeset=0。

做为开发用途的话，建议选择英文安装。

点击continue：

选中第一项“正常安装”：

等待10分钟左右，系统会自动安装完成。

四. 分区安装

与普通安装的区别，就是手动设置分区。在此步骤时，选择”其他安装”。

举个EFI分区设置实例，图片如下：

4个分区设置完成后，分区如下：

五. 双系统切换

此时Ubuntu系统安装完成后，接下来就是设置双系统切换。
F1 进入BIOS设置 ,把 UEFI/Legacy Boot 模式 设置成 Both。

以后每次开机重启，F12 然后通过 上下键选择你想要进入的系统即可。
